This repository contains different Finite Volume Method solvers
Different configurations for each solver are explained below
Solver 1: HEAT CONDUCTION / 1D / DIRICHLET B.C. on LEFT / DIRICHLET B.C. on RIGHT / Gauss-Seidel Iteration Method with Relaxation Factor
Solver 2: HEAT CONDUCTION / 1D / NEUMANN B.C. on LEFT / NEUMANN B.C. on RIGHT / Gauss-Seidel Iteration Method with Relaxation Factor
Solver 3: HEAT CONDUCTION / 2D / DICIHLET B.C. on TOP / DICIHLET B.C. on BOTTOM / DICIHLET B.C. on RIGHT / DICIHLET B.C. on LEFT / Gauss-Seidel Iteration Method with Relaxation Factor
Solver 4: HEAT CONDUCTION / 1D / NEUMANN B.C. on LEFT / NEUMANN B.C. on RIGHT / Gauss-Seidel Iteration Method with Relaxation Factor / WALL FUNCTIONS APPLIED
